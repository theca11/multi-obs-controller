REFRESH CAPTURE DEVICE
● Settings: Device, Delay
● No On/Off States
● Supported in Multi-Actions

Description

Disable and re-enable a video capture device

Settings

  • Device - Name of the device to refresh
  • Delay - Milliseconds to wait before re-enabling the capture device

States

Default
