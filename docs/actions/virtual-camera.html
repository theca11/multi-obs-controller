VIRTUAL CAMERA
● Settings: None
● On/Off States
● Supported in Multi-Actions

Description

Start/stop the virtual camera output

Settings

None

States

Active

Inactive

Note​

If you use multiple OBS instances in the same computer you can't enable the virtual camera more than once. This is a limitation of OBS itself and how the virtual camera resource is handled internally.
