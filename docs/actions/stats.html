STATS
● Settings: Stat
● No On/Off States
● Not Supported in Multi-Actions

Description

Display OBS Studio statistics

Settings

  • Stat - Statistic to display (check below)

States

Default

Configuring statistics to display​

You can configure the action to display as many statistics as you want (max total recommended 6 due to space limitations/font size). In the configuration panel, you can select the stat to display and the text color. Click on Add another stat to append a new stat to the key, and use the arrow keys to reorder stats. The stats for each OBS instance are concatenated in different sections, separated by a dotted line.

Resetting stats​

Long pressing a stats key will reset the stats related to missing/skipped/dropped frames. Note that this reset is local to the information shown in the Stream Deck and does not reset the OBS Studio stats shown in its own panel.

Available statistics​

General​

  • CPU Usage
  • Memory Usage
  • Available Disk
  • Active FPS
  • Average Frame Render Time
  • Missed frames (rendering lag)
  • Skipped frames (encoding lag)

Stream​

  • Active/Inactive status
  • Dropped frames (network congestion)

Recording​

  • Active/Inactive status
