CHAPTER MARKER
● Settings: Name
● On/Off States
● Supported in Multi-Actions

Description

Add a chapter marker to an active recording. As of OBS 30.2.0, the only file format supporting this feature is Hybrid MP4.

Settings

  • Name - Name of the marker to add

States

Active

Inactive
